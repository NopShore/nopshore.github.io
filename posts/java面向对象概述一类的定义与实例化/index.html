Java面向对象概述（一）：类的定义与实例化

2019-10-11
3分钟阅读时长
编程

面向对象三大特征

  • 封装 保护数据不被外部访问
  • 继承 对父类功能的扩展
  • 多态 一个对象的多种形态

类和对象

  • 类是对某一类事物的具有相同特征的抽象，是对象的模板。
  • 对象是类的具体化，实例化。
  • 一个类可以创建多个对象。

例如：动物是一个类，具有状态：身高、身长、体重、性别等等，行为：吃饭、睡觉等等。 猫、狗、兔、都可以是动物这个类的实例。但树就不行，因为树不具有动物这个类的特征。


类的定义与实例化

public class Demo{
    public static void main(String[] args){
        AnimalsCat bigCat = new AnimalsCat(); // 类的实例化(构造)
         // 设置大猫的状态
        bigCat.height = 50;
        bigCat.weight = 30;
        bigCat.age = 2;
        // 输出
        System.out.println("大猫的身长："+bigCat.height);
        System.out.println("大猫的体重："+bigCat.weight);
        System.out.println("大猫的年龄："+bigCat.age);
        System.out.print("大猫");
        bigCat.eat();

        System.out.println("=============================");

        AnimalsCat smallCat = new AnimalsCat(); // 类的实例化
         // 设置小猫的状态
        smallCat.height = 20;
        smallCat.weight = 13;
        smallCat.age = 0.3;
        // 输出
        System.out.println("小猫的身长："+smallCat.height);
        System.out.println("小猫的体重："+smallCat.weight);
        System.out.println("小猫的年龄："+smallCat.age);
        System.out.print("小猫");
        smallCat.eat();

        // bigCat(大猫)和smallCat(小猫)之间都是AnimalsCat的实例，
        // 这两个对象的状态和行为互不影响。
    }
}
// 定义猫这个类
public class AnimalsCat{
    int height; // 属性 身长
    int weight; // 属性 体重
    double age; // 属性 年龄

    // 方法
    public void eat(){ // 吃的食物
        if(this.age > 0.5) // 年龄大于0.5吃猫粮
            System.out.println("吃猫粮");
        else // 否则喝母乳
            System.out.println("母乳");
    }
    public void sleep(){
        System.out.println("sleep");
    }
}

bigCat(大猫)和smallCat(小猫)都是AnimalsCat的实例，他们都有各自的状态。 其中一个对象的状态和行为不会影响另外一个对象。
