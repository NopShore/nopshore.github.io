我在Linux上的常用软件（长期更新）

2019-09-26
2分钟阅读时长
软件相关

操作系统：ArchLinux (Arch死忠粉一枚)
桌面环境：KDE (plasma)


音乐：

  • netease-cloud-music (网易云音乐)
  • Listen1 (集成了网易云、虾米、QQ音乐、酷狗等各大平台的音乐)
  • amarok
  • ViPER4Linux (一款音效软件) (2019-09-12更新)
  • Petal (aur)(豆瓣)(2019-10-11更新)

视频播放：

  • VLC
  • Dragon Player

视频录制：

  • OBS Studio

截图：

  • flameshot (火焰截图)
  • Spectacle

聊天：

  • deepin-wine-qq (QQ)
  • deepin-wine-tim (TIM)
  • deepin-wine-wechat (微信)
  • mitalk (米聊)
  • telegram (电报)
  • irssi (基于命令行的irc聊天室工具)(2019-10-11)
  • Konversation (图形界面的irc聊天室工具)(2019-10-11)

RSS客户端：

  • Akregator (KDE自带RSS客户端)(2019-10-11)

Web浏览器：

  • Chrome
  • firefox
  • tor browser

虚拟机：

  • virtualbox

开发工具：

  • vscode
  • atom
  • jetbrains-toolbox
  • jetbrains-clion (C/C++)
  • jetbrains-pycharm (Python)
  • jetbrains-intellij idea (Java)
  • Android Studio (Android)
  • Qt Creator
  • Zeal (API文档查询工具)

Shell：

  • zsh
  • oh-my-zsh
  • bash

Markdown编辑器：

  • Typora
  • Vnote (markdown离线笔记软件) (已经不再使用，替换为Boostnote)
  • Boostnote

BTC钱包：

  • electrum

下载工具：

  • aria
  • qBittorrent
  • baidunetdisk (百度网盘)
  • baiduPCS-GO (基于命令行的第三方百度云多线程下载工具)

FTP：

  • filezilla

文件搜索：

  • catfish

办公：

  • wps-office
  • xmind-zen

小飞机：

  • qv2ray （2021-08-02更新）

手机管理：

  • kdeconnect (这个真的很强大)

护眼：

  • redshift

字体：

  • noto-fonts
  • noto-fonts-cjk
  • noto-fonts-emoji
  • noto-fonts-extra
  • ttf-arphic-uming
  • ttf-dejavu
  • ttf-wps-fonts
  • wqy-microhei
  • ttf-hack
  • ttf-monaco

输入法：

  • fcitx
  • fcitx-rime

美化：

  • latte-dock (kde桌面 dock栏)
  • sierrabreeze-kwin-decoration-git (kde桌面窗口装饰 MAC风)
  • yakuake (kde桌面 下拉式terminal)
  • cantalight (kde桌面主题)
  • la-capitaine-icon-theme (图标)

密码管理：

  • KeepassXC (2019-10-11更新)
