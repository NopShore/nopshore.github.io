Java面向对象概述（二）：封装

2019-10-11
2分钟阅读时长
编程
面向对象

封装的作用和优点

  • 减少耦合
  • 保护数据不被外界直接访问
  • 使外部传进来的数据得到效验
  • 保证了安全性

如何封装？

使用访问权限修饰符对数据进行权限控制 public protected default private


演示

public class Demo{
    public static void main(String[] args){
        AnimalsCat bigCat = new AnimalsCat();
//        bigCat.height = 50;  错误，height被设置成privete的，数据无法被外界直接访问
        bigCat.setHeight(50);  // 可以通过类中公开的方法对设置成private的进行访问
//        bigCat.setHeight(-50);   // 输出 错误 没有通过效验
        bigCat.setWeight(30);
        bigCat.setAge(2);

        System.out.println("bigCat的身高：" + bigCat.getHeight());
        System.out.println("bigCat的体重：" + bigCat.getWeight());
        System.out.println("bigCat的年龄：" + bigCat.getAge());
    }
}

public class AnimalsCat{
    // 封装属性
    private int height;
    private int weight;
    private double age;

    public void setHeight(int h){
        // 对数据进行效验 传进来的数据大于1则通过效验，否则提示错误，保证了安全性
        if(h > 0)
            height = h;
        else
            System.out.println("错误：身高不能小于1");
    }
    public void setWeight(int w){
        if(w > 0)
            weight = w;
        else
            System.out.println("错误：体重不能小于1");

    }
    public void setAge(double a){
        if(a > 0.0)
            age = a;
        else
            System.out.println("错误：年龄不能小于0.1");
    }

    public int getHeight() {
        return height;
    }
    public int getWeight() {
        return weight;
    }
    public double getAge() {
        return age;
    }

    public void eat(){
        if(this.age > 0.5)
            System.out.println("吃猫粮");
        else
            System.out.println("母乳");
    }
    public void sleep(){
        System.out.println("sleep");
    }
}
