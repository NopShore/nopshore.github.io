C语言的复杂声明

2019-08-09
4分钟阅读时长
编程

指针的定义与使用

int i = 5; // 声明一个int类型的变量，把5赋值给它。
int *p; // 声明一个指向int类型的指针。
p = &i; // 把i自身的地址赋值给指针*p。
printf("%d %p %p %p",*p,&p,p,&i);

最后的printf("%d %p %p",*p,p,&i); 解析：
第一个 *p 输出的是5（解引用得到p指向的地址存放的值）。
第二个 &p 输出的是指针p自身的地址。
第三个 p 输出的是指针p指向的地址，p = &i 所以p指向的是i的地址。
第四个 &i 输出的是i自身的地址。


多级指针

int i = 2;
int *p1 = &i;
int **p2 = &p1; // 声明一个二级指针p2，这个指针指向指针p1自身的地址。
printf(" %d\n %p\n %p\n %p\n %p\n",**p2,p2,*p2,p1,&p1);

最后的printf(" %d\n %p\n %p\n %p\n %p\n",**p2,p2,*p2,p1,&p1); 输出结果：

2  
0x7ffee13a2b08  
0x7ffee13a2b04  
0x7ffee13a2b04  
0x7ffee13a2b08

解析：
第一个 **p2 输出的是2（双重解引用 p2的值是p1的地址,p1的值是i的地址，i的值为2）。
第二个 p2 输出的是p1自身的地址 （p2指向指针p1的地址）。
第三个 *p2 输出的是i自身的地址 （p2解引用得到p1指向的地址，p1指向i，所以是输出i自身的地址）。
第四个 p1 输出的是i自身的地址 （p1指向变量i）。
第五个 &p1 输出的是p1自身的地址。


复杂声明

几个重要概念：

  • 指针数组 元素为指针的数组
  • 数组指针 指向数组的指针
  • 指针函数 返回值为指针的函数
  • 函数指针 指向函数的指针 优先级规则：
  • 优先级最高的是 声明中括号扩起来的内容
  • 其次是 后缀操作符 ( ) 、[ ]
  • 然后是 前缀操作符 *

简单的栗子：

int *arr[5];

arr先和后面的[ ]结合，说明这是一个数组，再与前面的*结合，说明这个数组里的元素是指针，
所以这是一个 指针数组。

int (*p)[5];

先看括号里的内容，p与*结合，说明这是一个指针，再与[ ]结合，说明这个指针指向数组，
所以这是一个 数组指针。

int *func(int);

func先于( )结合，说明这是一个函数，函数的参数为int，再与func前面的*结合，说明返回值是一个指针，
所以这是一个 指针函数。

int (*p)(int);

先看括号里的内容，p与*结合，说明这是一个指针，再与外面的括号结合，说明这个指针指向一个函数，外面的括号里是int，说明这个函数的参数是int类型的，
所以这是一个 函数指针。

复杂一点：

int *(*p)[5];

先看括号里的内容，p于结合，说明这是指针，再与[ ]结合，说明这个指针指向数组，再与第一个结合，说明这个数组里的元素是指针。
所以这是一个 数组指针，指向的数组的里面的元素应该是int *类型的。

int (*arr[5])(int);

先看arr所在的括号里的内容，arr[5]，arr先与[ ]结合，说明这是一个数组，再与结合，说明数组里的元素是指针，再与右边的括号结合，说明这个指针指向函数，这个函数的参数是int类型的。
所以这是 指针数组，数组里的指针应该指向函数。

int *(*arr[5])[5];

先看arr所在的括号里的内容，arr[5]，arr先与[ ]结合，说明这是一个数组，再与结合，说明数组里的元素是指针，再与右边的[ ]结合，说明这个指针指向数组，再与*结合，说明这个数组存放的元素应该是指针。
所以这是 指针数组，数组里的指针指向另一个指针数组。

复杂指针声明：

int (*(*func)[5][6])[7][8];

先看func所在的括号，func先于结合，说明这是指针，再与[5][6]结合，说明这个指针指向[5][6]的数组，再与第一个结合，说明这个数组的元素为指针，再与右边的[7][8]结合，说明这个指针指向[7][8]的数组，这个数组里的元素是int类型的。
简单说：这是一个 数组指针，指向一个[5][6]二维数组，数组的元素是指向[7][8]二维数组的指针。

int (*(*(*func)(int *))[5])(int *);

先看func所在的括号，func先于*结合，说明这是指针，再与右边的(int )结合，说明这个指针指向函数，函数的参数是int 类型的，再与第二个结合，说明这个函数的返回值是指针，再与[5]结合，说明这个函数返回的是[5]数组的指针，再与第一个结合，说明返回的这个数组指针指向另一个指针，这个指针指向参数是int *类型返回值是int类型的函数。
简单说：这是一个 函数指针，函数返回值是指向[5]数组的指针，数组的元素是一个函数指针。

int (*(*func[7][8][9])(int*))[5];

先看func所在的括号，func先于[7][8][9]结合，func前面是*，说明这是一个指针数组，再与(int *)结合，说明指向一个参数是int 的函数，再与第一个结合，说明这个函数的返回值是指针，这个指针指向[5]的数组，数组里的元素是int类型的。

简单说 ：这是一个 指针数组 ，这个数组的容量是[7][8][9]，数组元素是函数指针，函数的返回的值是指向[5]数组的指针。
