C语言冒泡排序算法

2019-08-26
2分钟阅读时长
编程

冒泡排序基本思想

不断比较数组中相邻的两个元素，让两个元素中较大的元素后移，经过一轮比较后，数值中最大的元素将会移动到数组末尾位置，第二轮将数组第二大的元素移动到数组倒数第二个位置。以此类推。

举个栗子

有一个 int arr[5] = {3, 2, 5, 4, 1} 数组，使它从小到大排序。


1.第一轮排序：
初始状态
3 2 5 4 1

比较 3 和 2，前者数值较大，后移 则变成：
2 3 5 4 1

比较 3 和 5，后者较大，不交换位置 不变：
2 3 5 4 1

比较 5 和 4，前者较大，交换后移 则变成：
2 3 4 5 1

比较 5 和 1，前者较大，交换后移 则变成：
2 3 4 1 5

经过第一轮排序，最大的数已经移动到末尾


2.第二轮排序：
第一轮排序结果： 2 3 4 1 5

比较 2 和 3，不交换位置 ：
2 3 4 1 5

比较 3 和 4，不交换位置 ：
2 3 4 1 5

比较 4 和 1，前者大，交换位置 ：
2 3 1 4 5

因为在第一轮比较中，已经将最大的数值移动到末尾，所以 4 和 5 不需要在进行比较。

经过第二轮比较，第二大的数移动到倒数第二的位置。


第三轮排序：
第二轮排序结果：
2 3 1 4 5

比较 2 和 3，不交换位置 ：
2 3 1 4 5

比较 3 和 1，交换位置 ：
2 1 3 4 5

因为在第上一轮比较中，已经将最大的数值移动到末尾，所以 3 和 4 不需要在进行比较。

经过第三轮比较，第三大的数移动到倒数第三的位置。


第四轮排序：
第三轮排序结果：
2 1 3 4 5

比较 2 和 1，交换位置 ：
1 2 3 4 5

经过第四轮比较，排序完成，第四大的数移动到倒数第四的位置，数值最小的元素则在一个位置。


实现方法

#include <stdio.h>

int* BubbleSort(int targetArr[],int sourceArr[],int size);
void ShowArr(int arr[],int size);

int main(int argc,const char * argv[]){

    int testArr[10] = {7,3,1,4,6,9,2,6,5,3};
    int arr[10];

    int *ptr = BubbleSort(arr,testArr,10);

    printf("排序前：\n");
    ShowArr(testArr,10);
    
    printf("排序后：\n");
    ShowArr(arr,10);

    return 0;
}

// 冒泡排序算法
int* BubbleSort(int targetArr[],int sourceArr[],int size){

    // 复制数组
    for(int i = 0;i < size;i++){
        targetArr[i] = sourceArr[i];
    }

    // size - 1 是需要比较的轮数，比如5个元素的数组，需要比较四轮
    for(int i = 0;i < size - 1;i++){ 

        for(int j = 0;j < size - 1 - i;j++){
            // size - 1 - i 是需要比较的元素个数，因为每轮都会完成一个元素的排序，
            // 所以每轮都会少一个元素需要排序，
            // 直到size - 1 - i 等于 0 时，则说明整个数组的元素排序完成。
            if(targetArr[j] > targetArr[j+1]){
                int temp = targetArr[j];
                targetArr[j] = targetArr[j+1];
                targetArr[j+1] = temp;
            }
        }
    }

    return targetArr;
}

// 打印数组内容
void ShowArr(int arr[],int size){

    for(int i = 0;i < size;i++){
        printf("%d ",arr[i]);
    }
    printf("\n");
}

输出结果:
排序前：
7 3 1 4 6 9 2 6 5 3
排序后：
1 2 3 3 4 5 6 6 7 9
